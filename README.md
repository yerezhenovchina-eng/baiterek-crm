# Baiterek CRM System

Корпоративная система управления советом директоров и структурами организаций.

## 🚀 Возможности

- **Управление организациями** — Советы директоров, Правления, Комитеты
- **Управление людьми** — члены СД, сотрудники, ответственные лица
- **Документирование** — заседания, повестки, решения ЕА
- **KPD система** — личные КПД сотрудников и групповые
- **Агент мониторинга** — отслеживание изменений составов
- **Экспорт/Импорт** — Excel шаблоны, PDF справки

## 📋 Структура

```
.
├── main.py                 # FastAPI приложение
├── gen_org_pdf.py         # Генератор PDF отчётов
├── kpd_import_export.py   # Импорт/экспорт КПД
├── database.py            # Работа с БД
├── templates/             # HTML шаблоны
├── static/                # CSS, логотипы, шрифты
└── requirements.txt       # Зависимости Python
```

## 🔧 Установка

```bash
pip install -r requirements.txt
python main.py
```

Затем откройте http://localhost:8000
